\section{Callouts Modernizados}

\begin{calloutTip}[Consejo Profesional]
Los nuevos colores vibrantes mejoran la legibilidad y el contraste visual sin perder la identidad institucional.
\end{calloutTip}

\begin{calloutWarning}[Nota Importante]
Todos los elementos mantienen los colores institucionales base, solo se han agregado variantes más vibrantes para acentos.
\end{calloutWarning}

\section{Combinaciones Recomendadas}

\subsection{Para Títulos y Encabezados}
Use \textbf{accentGuinda} o \textbf{accentDorado} para mayor impacto visual.

\subsection{Para Resaltados en Texto}
Use \texttt{\\highlightSoft} con fondos sutiles para no distraer del contenido.

\subsection{Para Elementos Interactivos}
Use \texttt{\\badge} y \texttt{\\tag} con colores vibrantes para llamar la atención.

\subsection{Para Notas y Advertencias}
Use \texttt{calloutTip} y \texttt{calloutWarning} con los colores modernizados.
